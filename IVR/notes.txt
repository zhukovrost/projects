Формулировка темы проекта - Сайт-конструктор физических тренировок.

Формулировка актуальности проекта: 
1. Многие спортсмены сталкиваются с проблемой поиска программы тренирок для поддержания или улучшения своей физической формы в домашних условиях. Готовые программы не всегда подходят человеку. Я создал сайт-приложение, где пользователь сможет не только создать собственную программу тренировок на нужные ему группы мышц из библиотеки тренировок, но и отслеживать свой прогресс.
2. Данное приложение поможет человеку начать заниматься спортом, составить программу тренировок, которая ему под силу.

Формулировка новизны проекта - создание сайта-конструктора физических тренировок (Уровень модификации).

Постановка цели проекта - Целью данного проекта является создание сайта-конструктора физических тренировок.

Постановка задач проекта:
1. Построить структуру сайта на всех страницах с помощью языка гипертекстовой разметки документов HTML.
2. Создать базу данных с помощью свободной реляционной системы управления базами данных MySQL.
3. Написать код для взаимодействия с базой данных с помощью языка программирования PHP.
4. Сделать дизайн каждой страницы с помощью языка программированя JavaScript и языка описания внешнего вида документа CSS.
5. Выгрузить сайт и базу данных в сеть Интернет.
